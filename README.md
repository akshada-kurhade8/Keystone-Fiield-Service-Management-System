# Keystone Field Service Management System

## Project Overview

Keystone Field Service Management System is a full-stack web application developed to manage field service operations. It allows administrators to manage customers, sites, employees, work orders, attendance, salaries, leaves, and reports through a secure dashboard.

## Technologies Used

### Backend
- Java 21
- Spring Boot
- Spring Security
- JWT Authentication
- PostgreSQL
- Maven

### Frontend
- React.js
- Vite
- Bootstrap
- Axios

## Features

- User Authentication (JWT)
- Role Based Access Control (RBAC)
- Customer Management
- Site Management
- Employee Management
- Work Order Management
- Attendance Management
- Salary Management
- Leave Management
- Reports Dashboard

## Project Structure

```
backend/
keystone-frontend/
```

## How to Run

### Backend

1. Open backend project.
2. Configure PostgreSQL database.
3. Update application.properties.
4. Run BackendApplication.java.

### Frontend

```bash
npm install
npm run dev
```
